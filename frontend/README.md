# GitHealth App

Single-page React + Vite dApp for the GenLayer `GitHealth` intelligent contract.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Confirm contract settings in `src/constants.js` (especially `CONTRACT_ADDRESS`).

3. Ensure GenLayer Studio RPC is running at `http://localhost:4000/api`.

4. Start the app:

```bash
npm run dev
```

## Build

```bash
npm run build
```
